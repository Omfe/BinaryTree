Detailed Description

A SubtreeView draws nothing itself (unless showsSubtreeFrames is set to YES for the enclosingTreeGraph), but provides a local coordinate frame and grouping mechanism for a graph subtree, and implements subtree layout.

Method Documentation

- (id <PSTreeGraphModelNode>) modelNodeAtPoint: (CGPoint)  p

Returns the visible model node whose nodeView contains the given point "p", where "p" is specified in the SubtreeView's interior (bounds) coordinate space.

Returns nil if there is no node under the specified point. When a subtree is collapsed, only its root nodeView is eligible for hit-testing.
